# backsteros-admin

**Ops / API dashboard** — logs, sync behavior, service health, storage stats, API keys.

| Surface | URL |
| --- | --- |
| Web | `https://backsteros.com/admin` |

**Not** for creating tasks or editing markdown — that is [`backsteros-app`](../backsteros-app/) (`/app`).

Online-first; separate codebase from product app (lighter UI, read-mostly).

**Status:** Not started — Phase 3b (after API + sync exist to display).
